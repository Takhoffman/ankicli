# Learning Plans

Use this reference when the task is to design a goal-driven learning plan around a deadline, a time budget, or a specific target domain.

## Typical use cases

- Travel preparation with a fixed date and daily time budget
- Anime or media immersion around a specific show or genre
- Exam preparation, including medical-student workflows
- Adaptive daily review and new-card planning based on what the learner already knows

## Rules

1. Start with the real goal, deadline, and available daily time.
2. Inspect existing deck coverage before proposing new cards.
3. Optimize for practical payoff, not abstract coverage.
4. Prefer a staged plan: inspect, propose, then create or update cards after approval when quality matters.

## Workflow shape

1. Inspect existing deck state.
2. Identify gaps relevant to the real goal.
3. Propose a plan that fits the time budget.
4. Create or update cards only after the content direction is clear.
